# Лабораторная работа №2, задание 3 (вариант 5)

## Overview
Программа выполняет две задачи:
1. Поиск чисел, являющихся полными квадратами, в массиве случайных чисел.
2. Работа со структурой NOTE1 (9 записей): ввод, сортировка по фамилии, вывод по месяцу рождения.
